# Data Security and Confidentiality

A comprehensive study on data protection techniques, encryption methods, and regulatory compliance. Includes a full research report, presentation, and an interactive Caesar cipher application.

## Overview

This project explores the critical importance of data security and confidentiality in the digital age, covering encryption techniques, threat analysis, and regulatory frameworks.

## Topics Covered

**Encryption Methods:**
- Symmetric Encryption (AES, DES)
- Asymmetric Encryption (RSA)
- Classic Methods (Caesar Cipher)

**Threats & Protection:**
- Cyberattacks (Phishing, DDoS, SQL Injection)
- Ransomware
- Internal Threats

**Regulations:**
- GDPR (General Data Protection Regulation)
- Cloud Act

## Repository Content

- **Report.pdf**: Complete research report with methodology and analysis
- **Presentation.pptx**: PowerPoint presentation on data security
- **Code/**: Source code
  - `caesar_cipher_application.py`: Interactive Caesar cipher application (Python + Tkinter)

## Key Concepts

- **Confidentiality**: Restricted access to authorized users only
- **Integrity**: Protection against unauthorized modifications
- **Availability**: Continuous accessibility of data
